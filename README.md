## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/murilofelpeto/connection_pool.git
```

Entre no diretório do projeto

```bash
  cd connection_pool
```

Instale as dependências

```bash
  mvn clean install
```

Inicie a applicação

```bash
  run the main class (ConnectionPool)
```


## Stack utilizada

**Back-end:** Java, Apache http client
